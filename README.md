# TEKMIZ — E-LEARNING PLATFORM

Tekmiz is a MERN stack based e-learning platform designed to help students manage study resources like videos, PDFs, and PPTs using playlists, so last-minute exam preparation doesn’t become chaotic.

## Features
- User authentication (signup & login)
- Playlist-based resource management
- Upload and organize videos, PDFs, and PPTs
- Easy access to all study materials in one place
- Student-friendly and responsive UI
